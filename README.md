# Snes9x++
*Snes9x++ - Super Nintendo Entertainment System (TM) emulator with Kaillera*

This is the official source code repository for the Snes9x++ project.

## Features
 
- **Old font restored** - The classic font has been restored for authentic nostalgia
- **Kaillera added** - Kaillera client support for multiplayer gaming
- **SRAM/savestate/hacks automatic sync through Kaillera** - Automatically syncs SRAM, savestates, and hacks (like Super FX overclock) during Kaillera sessions
- **Netplay fixed** - Netplay functionality has been fixed
- **On-screen chat for Kaillera and netplay** - Built-in on-screen chat for both Kaillera and netplay sessions
- **Kaillera and Netplay won't overwrite your SRAM if you aren't hosting** - Protection against accidental SRAM overwrites when joining games
- **Automatic SMV recording** - Automatic SMV recording for netplay, Kaillera, and offline gameplay
- **SMV to AVI easy conversion** - Easy conversion of SMV movies to AVI format
- **Can save patched ROMs** - Ability to save ROMs that have been patched
- **Automatic multitap type selection on Kaillera** - Automatic multitap type selection during Kaillera sessions
- **Sync improvements** - Netplay host can select the input type for the session, more desync prevention
